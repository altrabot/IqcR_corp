# IQC Generator

Website untuk membuat screenshot chat WhatsApp versi iPhone yang 100% realistis.

## Fitur

- Membuat screenshot chat WhatsApp dengan tampilan iPhone asli
- Sistem token sekali pakai (TCR)
- Tampilan responsif dan user-friendly
- Simulasi status bar iPhone dengan baterai, jaringan, dan waktu
- Reaksi emoji otomatis

## Cara Menggunakan

1. Buka website IQC Generator
2. Login dengan nama pengguna dan token TCR
3. Isi form dengan teks chat, jam, nama jaringan, dan persentase baterai
4. Klik "Generate Screenshot" untuk membuat screenshot
5. Unduh atau bagikan hasil screenshot

## Sistem Token (TCR)

- Token format: `TCR-xxxxx-XX` (xxxxx = kode acak, XX = jumlah token)
- Token baru di-generate otomatis setiap 24 jam
- Token bersifat sekali pakai
- Jika token habis, pesan token baru melalui tombol "Order Token"

## Teknologi

- HTML5, CSS3, JavaScript ES6
- LocalStorage untuk penyimpanan data
